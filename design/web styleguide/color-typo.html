Logo Color/Typography Imagery/Icon

I don't have a strong logical argument for picking these colors other than these colors looks nice together and the shades complement each other well.

The website I used to pick these is colormind.io.

Usage

Darker colors will primarily be used as backdrop.

Lighter colors (blue to green) will be used for buttons, text highlighting/hover states.

Menu colors will be green/darkgreen depending on theme mode.
